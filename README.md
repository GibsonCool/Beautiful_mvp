# Beautiful_mvp

一个自己用的慢慢完善的mvp架构

*Add it in your root build.gradle at the end of repositories:*
```
allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

*Step 2. Add the dependency*
```
dependencies {
	        compile 'com.github.GibsonCool:Beautiful_mvp:2.0'
	}
```
